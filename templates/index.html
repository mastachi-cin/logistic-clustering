Logistic clustering App

About our app

Routify is a data driven logistics solution that provides a simple yet powerfull way for users to decide on optimal solutions for logistic problems.

Our Beta is based on clustering locations based on KMeans and optimal number of routes according to certain user inputs.

Destinations: Number of packages to be delivered to different locations.

Seed: Setting up the random state (this number sets a start point for random location selection).More info here

Max cargo per vehicle: Vehicle cargo capacity in number of packages.

Number of vehicles: Available fleet size.

For this first MVP, Routify has a database with over a thousand dummy locations within Monterrey Metro Area. All of this locations come from a real world logistics company that sponsored our project with data.

Future updates will make this the 'go to' place for logistics operators.
